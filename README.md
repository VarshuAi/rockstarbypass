# Vortex Launcher

A lightweight, custom game launcher built with Electron.

## Features

* Custom Emerald Obsidian theme with glassy UI components.
* Integrated account management.
* Built-in local configuration.

## Development

To run the app in development mode:
```bash
npm install
npm start
```

To build the distribution executable:
```bash
npm run build
```
